This project contains exercises to practice fundamental JavaScript concepts. Follow the instructions for each exercise precisely. The automated tests will check your work against these exact requirements.

## Exercises

### 1. Variables and Data Types
- **Task 1.1**: Create a variable named `studentName` and assign it the string value `"John Doe"`.
- **Task 1.2**: Create a variable named `studentAge` and assign it the number value `20`.
- **Task 1.3**: Create a variable named `isEnrolled` and assign it the boolean value `true`.

### 2. Functions
- **Task 2.1**: Create a function named `greet` that takes one parameter `name` and returns the string `"Hello, [name]!"` (replace `[name]` with the actual parameter value).
- **Task 2.2**: Create a function named `add` that takes two parameters `a` and `b` and returns their sum.
- **Task 2.3**: Create a function named `isEven` that takes one parameter `number` and returns `true` if the number is even, `false` otherwise.

### 3. Arrays
- **Task 3.1**: Create an array named `colors` containing exactly three strings: `"red"`, `"green"`, and `"blue"`.
- **Task 3.2**: Create a function named `getFirstElement` that takes an array as a parameter and returns the first element of that array.

### 4. Objects
- **Task 4.1**: Create an object named `car` with three properties: `brand` (string value `"Toyota"`), `model` (string value `"Camry"`), and `year` (number value `2020`).
- **Task 4.2**: Create a function named `getFullName` that takes an object with properties `firstName` and `lastName`, and returns a string in the format `"[firstName] [lastName]"`.

## Getting Started
1. Clone this repository.
2. Open `script.js` and complete the tasks.
3. Run the tests to check your progress.

## Testing
Run the tests to check your progress. Each task has its own test suite.
```bash
npm install
npm test
```
